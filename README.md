```bash

```

### **2. 📦 Установка зависимостей:**
```bash
pip install -r requirements.txt
```

### **3. 🔑 Настройка API (опционально):**
```bash
# Создайте .env файл с ключами:
WEATHER_API_KEY=ваш_ключ_openweathermap
CALENDAR_API_KEY=ваш_ключ_calendarific  
OPENAI_API_KEY=ваш_ключ_openai
```

### **4. 🚀 Первый запуск:**
```bash
# Проверка готовности системы
python3 main.py check-apis

# Список доступных ресторанов
python3 main.py list

# Полный анализ ресторана - ВСЁ В ОДНОМ!
python3 main.py restaurant "Ika Canggu"

# Полный анализ рынка
python3 main.py market
```
