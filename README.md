# 🎯 MuzaQuest Mini App - Система Бизнес-Аналитики для Ресторанов

## 🚀 Описание

**MuzaQuest Mini App** - это интеллектуальная система бизнес-аналитики для ресторанов, которая автоматически анализирует продажи, выявляет причины изменений и предоставляет конкретные рекомендации для улучшения бизнеса.

### 🎯 Основные возможности

- **📊 Анализ продаж** - Детальный анализ изменений продаж с выявлением причин
- **💡 Умные рекомендации** - Конкретные действия для улучшения показателей
- **🔍 Анализ факторов** - Влияние рекламы, погоды, праздников, отмен
- **📈 Отчеты и тренды** - Недельные отчеты и краткие сводки для руководства
- **🧪 Тестирование гипотез** - Статистическое подтверждение бизнес-решений

## 🛠️ Установка и запуск

### Требования
- Python 3.8+
- SQLite3

### Быстрый старт

```bash
# Клонирование репозитория
git clone https://github.com/muzaquest/muzaquest-mini-app.git
cd muzaquest-mini-app

# Установка зависимостей
pip install -r requirements.txt

# Запуск анализа
python main.py analyze --restaurant "Canggu Surf Cafe" --date "2023-06-15"
```

## 📋 Доступные команды

### 📊 Анализ конкретного случая
```bash
python main.py analyze --restaurant "Название ресторана" --date "2023-06-15"
```

### 📈 Недельный отчет
```bash
python main.py weekly --restaurant "Название ресторана" --weeks 4
```

### 🎯 Краткий отчет для руководства
```bash
python main.py summary --restaurant "Название ресторана"
```

### 🧪 Тестирование гипотез
```bash
python main.py test --restaurant "Название ресторана" --hypothesis "реклама эффективна"
```

### 📋 Список ресторанов
```bash
python main.py list
```

### 🤖 Обучение модели
```bash
python main.py train --model-type random_forest
```
